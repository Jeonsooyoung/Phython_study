## 1. Data Type
- 변수 선언 :  (number) a = 2, b = 3.12,
              (text) c = "3", 
              (boolean) c = True,  d = False 
              (none) e = None (존재하지 않음)
              
- 타입 표시 : print(type + (변수명) )
- variable type : 숫자, 문자, 참.거짓, null 값
- python 변수 컨벤 션 : snake case (ex) super_long_variable
